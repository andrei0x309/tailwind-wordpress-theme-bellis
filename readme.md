## Bellis WP Theme

Fork of A309 WP Theme

### Is using:
- tailwind css
- laravel mix

### Features:
- AMP compatible
- Unregisters Jquery and other stuff from WP
- Adds Custom REST API
- Vanila JS
- Mobile First
- Lightweight + Performance
- Light / Dark Mode
- Native lazy load
- Dynamic feature image request size from CDN based on display size
